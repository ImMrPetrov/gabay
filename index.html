GABAY

A Mobile Application For Learning The Tagalog Language For English Speaking Kids
Who is Gabay?

Gabay is an Eagle that is willing to teach children Tagalog languages.

Purpose of Gabay

An engaging and interactive mobile app designed specifically for English-speaking children eager to learn the Tagalog language. With colorful visuals, fun games, and immersive activities, kids embark on an exciting journey to grasp Tagalog vocabulary, phrases, and cultural nuances effortlessly.

App Features

Gabay provides features that helps kids into learning Tagalog Language in an interactive ways

Phonics

The application aims to help users learn and improve their phonics skills. By exploring letter-sound relationships, users enhance their reading and speaking abilities, gaining the foundational knowledge needed to recognize and pronounce words accurately.

User Profile

The app offers the capability for parents to create individual user profiles, catering to households with multiple children. The feature ensures that parents are constantly informed about their child's performance and achievements within the application.

Educational Videos

The application features educational videos and audio that cover various topics to facilitate learning.

Games

Consists of Matching Game, Counting Game, Sorting Game, and Shape Recognition Game
